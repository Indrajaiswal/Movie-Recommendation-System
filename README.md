## 🎬 Movie Recommendation System

A Movie Recommendation System built using Python, KNN Algorithm, and Streamlit that suggests similar movies based on user selection or genre preferences.

This project uses the IMDB 5000 Movie Dataset and integrates TMDb API to fetch movie posters, ratings, release dates, and overview.


## 🚀 Features

🔍 Search or select a movie to get recommendations

🎥 Displays movie posters using TMDb API

⭐ Shows TMDb and IMDb ratings

📅 Displays release date and movie overview

🎛️ Choose number of recommendations (5–20 movies)

🎨 Attractive Streamlit dashboard with CSS styling

📊 Genre-based recommendation option

📦 Uses K-Nearest Neighbors (KNN) algorithm


## 🛠️ Technologies Used

Python

Streamlit

KNN Algorithm

Pandas & NumPy

BeautifulSoup (for IMDb scraping)

TMDb API

HTML & CSS

## 📊 Dataset

This project uses:

IMDB 5000 Movie Dataset

Movie metadata including genre, ratings, and features.
